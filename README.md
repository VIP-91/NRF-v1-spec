   🔧 NRF-spec is a specification for NetEase Rental Functions, make mcfunctions easier to use. It is designed for NetEase's 1.12.2 version server.

   ⚡️ The specification aims to make functions easier to maintain and extend, while also specifying the positions of some special instructions, which facilitates quick location and modification. It also facilitates integration with specific obfuscators for obfuscation.
